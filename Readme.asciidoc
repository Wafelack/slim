== SLIM

SLeep IMproved - sleep(1) alternative written in ANSI C.

=== Installing

You can set the +MAKE_SYMLINK+ variable to +0+ if you do not want slim to be linked to +${PREFIX}/sleep+.

[source,bash]
----
doas make install
----
